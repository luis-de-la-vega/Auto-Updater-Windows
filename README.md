# 🚀 LuisDev AutoUpdater

Aggiorna automaticamente tutte le applicazioni installate su Windows utilizzando **Winget**, in modo semplice, veloce e senza intervento manuale.

---

## ✨ Descrizione

**LuisDev AutoUpdater** è uno script `.bat` leggero progettato per mantenere il tuo sistema sempre aggiornato senza complicazioni.

✔ Nessuna configurazione
✔ Avvio automatico
✔ Interfaccia CLI moderna
✔ Nessun log (versione clean)

Perfetto per utenti e sviluppatori che vogliono un aggiornamento rapido e senza distrazioni.

---

## ⚙️ Requisiti

* Windows 10 / 11
* Winget installato
* Terminale compatibile con ANSI (consigliato Windows Terminal)

---

## 📦 Installazione

1. Scarica il file:

   ```
   LuisDev-AutoUpdater.bat
   ```

2. Posizionalo dove preferisci (es. Desktop)

3. Esegui il file con doppio click

---

## ▶️ Utilizzo

Avvia semplicemente lo script:

```bash
LuisDev-AutoUpdater.bat
```

Lo script:

1. Controlla la presenza di Winget
2. Analizza il sistema
3. Aggiorna automaticamente tutte le applicazioni
4. Mostra lo stato in tempo reale

---

## 🎨 Interfaccia

* 🟢 `[OK]` → operazione completata
* 🔴 `[ERRORE]` → problema rilevato
* 🟡 `[*]` → operazione in corso
* 🔵 `[>>]` → processo principale

---

## ⚠️ Avvertenze

* Gli aggiornamenti automatici possono causare incompatibilità con alcuni software
* Non viene effettuato alcun backup
* Alcuni aggiornamenti potrebbero richiedere privilegi di amministratore

👉 Usa con consapevolezza, soprattutto in ambienti di sviluppo
